Get Running
⚡ Quick Start — 3 Commands

No API keys, no cloud setup. Runs fully offline after a one-time model download (22 MB).

# 1. Clone the repo git clone https://github.com/AnkitSingh36/rag-universe cd rag-universe # 2. Install dependencies pip install -r requirements.txt # 3. Launch the app streamlit run app.py # Opens at http://localhost:8501

✅ What happens on first run

  • Downloads a 22 MB embedding model (once only)
  • All demos use built-in sample data
  • Runs 100% locally — no internet needed after setup

💡 Optional: Richer Answers with Ollama

Install Ollama and any local model (Llama 3, Mistral…) for real LLM-generated answers instead of demo templates. The app auto-detects Ollama — no config needed.
