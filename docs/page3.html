🧪 How to Test Each RAG Type

Clone the repo and follow these tests to see each architecture's unique behavior.

Test 1
🔵 Naive RAG — Baseline
Topic: 🚀 Space Exploration
RAG: Naive RAG
Ask: "Who first walked on the Moon?"
✅ Look for: yellow query-word highlights in chunks, relevance score bars
Test 2
🟣 Hybrid RAG — Keyword + Semantic
Topic: 💻 Technology & AI
RAG: Hybrid RAG
Ask: "What is RAG?"
✅ Look for: chunks labeled "hybrid (both)" — found by BM25 AND vector search
Test 3
🟢 Conversational RAG — Memory
Topic: 🐘 Animals & Wildlife
Ask Q1 → Run → Then ask "How much do they eat?"
✅ Look for: "Conversation Memory" section showing stored turns
Test 4
🔮 Self-Reflective RAG
Ask a question NOT in the document
e.g., "What is quantum tunneling?"
✅ Look for: reflection steps → query expansion → low-confidence flag
Test 5
✅ Corrective RAG — Grader
Ask something partially covered in the doc
See how confidence is graded
✅ Look for: CORRECT / AMBIGUOUS / INCORRECT classification in answer
Test 6
🔀 Adaptive RAG — Auto-Route
Ask 3 questions of different complexity:
"What year?" → "What is ISRO?" → "Compare and contrast…"
✅ Look for: [Adaptive routing: Used X RAG] at start of each answer
